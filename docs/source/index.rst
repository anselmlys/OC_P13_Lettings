Documentation du projet OC Lettings
===========================================

Orange County Lettings est un site web développé dans le cadre de la formation OpenClassrooms - Développeur d'application Python.

L'objectif principal du projet est la mise en place d'une pipeline CI/CD automatisant :

* les tests
* la création d'une image Docker
* le déploiement de l'application Django sur Render
